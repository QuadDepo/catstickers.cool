# catstickers.cool

![A collage of cat meme stickers with thick white sticker borders.](./public/og-image.png)

Tiny cats, silly cats, cursed cats, cute cats. Just a ridiculous pile of cat
stickers.

Grab a cat, save it, and send it wherever the group chat needs emotional
support. WhatsApp, Telegram, Discord, iMessage, Slack, your desktop sticker
folder, wherever.

## Stickers for chats

- Meme cats for WhatsApp replies.
- Cute cats for harmless nonsense.
- Cursed cats for when words are too normal.
- Downloadable sticker images, ready to throw into whatever app you use.

## Develop

```bash
pnpm install
pnpm dev
```
